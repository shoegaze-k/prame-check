# param-check

<br>

URLのパラメータに不備が無いかチェックします。

Check the URL parameters for any deficiencies.

<br>

---

<br>

## setting

チェックのバリデートの設定は「popup.js」を編集してください。　

Edit "popup.js" to set the validation of the check.
